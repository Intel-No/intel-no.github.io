安装Burp Suite

在Windows上安装

文件下载：点我下载

步骤一：下载来自我上传的文件（你会得到步骤二的图片中的几个文件）

文件说明：old文件夹中含老版本的burp suite（为windows7老系统准备的），正常按照我下面的说明，win10/11的安装最新版的即可（这里我使用window11来进行演示）

步骤二：双击此文件，安装Java坏境
步骤三：开始安装

点击继续（选择文件位置，这里我建议可以更换一个自己记得住的路径）

完成安装（这里点击关闭即可）

步骤四：配置坏境变量

依次打开：设置，系统，系统信息，高级系统设置

选择环境变量选项

依次选择下面图片中的选项

*注意：这里的变量名，建议固定为 JAVA_HOME（不是必须，但强烈推荐，可以为后续省去很多不必要的麻烦）

最后全部选择“确定”选项

步骤五：配置用户环境变量

按照如下步骤来就行，不要跳步（如果你前面的名称和我一样，那就按照我图片的步骤来即可）

步骤六：将这个环境变量移动至顶部

不出意外的话，你配置完，在终端中输入

javac

因该能在显示这个界面

至此，坏境变量部分即为配置成功

步骤七：开始安装Burp suite

右击这个burploader.jar文件，打开方式选择Java(TM)Platform SE binary

在弹出的窗口中把这两个勾打上，并点击Run

如果不出意外，你会自动打开burp suit的界面（如下图所示）

还有种情况（那就是和我一样，没打开，没反应）

那就和我一样，手动再次双击这个burploader.jar文件，打开这个界面，也是同样的效果（要保持这个之前的页面不要关，要有两个窗口）

将左侧的文本复制，粘贴到右侧，随后点击NEXT

继续按照以下步骤操作

粘贴完成之后，再次点击NEXT，即可看到激活成功（如下图所示）

点击Finish结束即可

步骤八：下载证书

按照下图的步骤配置即可

选择下图的框选区域

命名要求必须为.der结尾

点击保存即可（下图是我的路径，可供参考）

步骤九：配置证书

打开浏览器，搜索证书（这里我以最普遍的Edge浏览器来示范，其他的浏览器也是同理）

打开这个依次打开如下的选项

选自刚刚保存的证书路径，找到证书

中间证书配置也是如此（和上面的步骤一致，我就不再演示了）

配置好后如下图所示

步骤十：安装浏览器拓展

安装浏览器拓展： proxy switchyomega

点击右上角的“拼图”，打开拓展（可以固定到这里，方便开关）

按照下图步骤配置插件

继续按照下图，来进行配置

最后选择左下角的“应用选项”，保存即可
